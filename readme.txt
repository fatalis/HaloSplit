HaloSplit is a LiveSplit (livesplit.org) component for Halo: Combat Evolved.

Features:

Automatically start/stop the timer when player control is gained/lost.
Automatically split when the level changes.
Automatically reset when the first level starts.

Requirements:

Halo CE 1.0 or 1.01
LiveSplit 1.3+

Install:

Extract LiveSplit.HaloSplit.dll to your LiveSplit\Components folder. Restart
LiveSplit. Add HaloSplit in LiveSplit's Layout Editor. It will not be visible
on your splits and will run in the background.
